IShape interface

A Shape is a class that implements the IShape interface.

[TypeConverter(typeof(ShapeConverter))]
public interface IShape : IDisposable
{
    [Browsable(false)]
    string Name { get; }
    bool Visible { get; set; }
    event EventHandler Changed;
    void Draw(Graphics g);
}

The most important method is Draw. It draws the shape on the clock's dial.

The Changed event is used to announce the clock that the one of the Shape's parameter was change and that it should repaint itself.
