Features

This is the section where I prise my control:

  • Decoupled design. The actor (that updates the control), tha data (that is displayed) and the view (the way the clock looks) are decoupled.
  • Customizable appearance. (Using custom implementations of the IShape interface.)
  • The ability to display a custom time (not only the system's time) using a custom implementation of the ITimeProvider.
  • Scalable GUI
  • Easy to use.
