The Shapes

A Shape is a class that draws a graphical element on the clock. To be a Shape a class must implements the IShape interface.

The clock contains three types of shapes:

  • Background Shapes - They draw the dial's background.
  • Angular Shapes - They are repeated at different angles around the clock, like the ticks and the numbers.
  • Hand Shapes - They draw the hands that display the time.

ShapeBase abstract class is the default implemnentation of the IShape interface. When creating a new Shape, it is recomanded to derive from this class or one of its descendents, rather then implementing the IShape members from scratch.
