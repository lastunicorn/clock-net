The Timer

The time displayed by the AnalogClock control can be set manually or using a timer.

To manually set the time, set the Time property of the AnalogClock control. By doing that, the clock will display the specified time indefinitely until you set the Time property with another value.

Another way to display a time in the AnalogClock control is by using a timer and a ITimeProvider object. The timer periodically updates the clock with a new value. But the value that is used comes from the ITimeProvider object.

The timer that can be used is the System.Windows.Forms.Timer component that can be found in the "Components" category of the "ToolBox" window.
