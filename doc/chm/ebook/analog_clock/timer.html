The Timer

The timer periodically updates the clock with a new time value.

The value displayed by the timer can be obtained internally by ClockNet (using the DateTime class) or from an external source, the TimeProvider property.

By default, ClockNet uses the internal mechanism to obtain the value to be displayed. You can configure ClockNet to display the local time, or the utc time decalated with a specified offset. For that, you need to set the UtcOffset property. If this property is null, the local time is displayed. If this property contains a value, the utc time is displayed to which the specified value is added. To display exactely the utc time, set the property to "00:00:00".

To use an external Time Provider, you just need to set the TimeProvider property with an instance of ITimeProvider and ClockNet will automatically use it.

More details about the Time Provider can be found here.

The timer that can be used is the System.Windows.Forms.Timer component. It is found in the "Components" category of the "ToolBox" window.
