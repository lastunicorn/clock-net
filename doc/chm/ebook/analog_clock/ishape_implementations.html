IShape Implementations

ShapeBase abstract class is the default implemnentation of the IShape interface. When creating a new Shape, it is recomanded to derive from this class or one of its descendents, rather then implementing the IShape members from scratch.

Two other classes are extending the ShapeBase class:

  • VectorialShapeBase - It is also an abstract class and offers support for drawing a vectorial image.
  • ImageShape - Draws an image on the clock.

Before creating a new custom Shape class check the IShapes Hierarchy to find the appropriete base class to extend.
