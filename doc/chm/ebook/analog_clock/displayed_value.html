The Displayed Value

The value displayed by the clock can be updated in two ways:

  • Manually - By setting the Time property. This is a static value, and the clock will display the respective time value indefinitely in time, until a new value is set (manually or by a timer).
  • Periodically - Using a timer. The timer will update the clock with time values obtained internally from the AnalogClock itself or from an external Time Provider.
