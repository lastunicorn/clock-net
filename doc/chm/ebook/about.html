About ClockNet

What is ClockNet?

ClockNet is a .net library that contains an analog clock control.

Who can use it?

The intended target users are the .net developers that need such a control in their applications.

Current version

ClockNet 2.0 alpha2 (2.0.0.21744)

Author

Dust in the Wind

Other informations
