Change Log

ver 2.0 alpha2

  • Bug fix - In design time, the control was not initializing the default shapes. - Fixed
  • I added some aditional explanations about how to use the control in the readme.txt file.

ver 2.0 alpha

  • The displayed time is now retrieved from an ITimeProvider instance.
  • The UI is created like a puzzle using IShape objects.
  • I created a lot of IShape objects that draws different types of graphic elements on the clock.
  • I created a number of predefined shape sets that can be used programatically to easily change the aspect of the clock.
Obs! In the future versions, the hierarchy of the IShape objects may change.
