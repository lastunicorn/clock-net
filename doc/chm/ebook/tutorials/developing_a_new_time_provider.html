Developping a New Time Provider

A Time Provider is the way in which you can provide an external custom time to be displayed by the timer. To create a new Time Provider you need to do the folowing steps:

  1. Implement the ITimeProvider interface:

    public class MyTimeProvider : ITimeProvider
{
    ...
}

    Or you can extend the TimeProviderBase abstract class that already implements the ITimeProvider interface:

    public class MyTimeProvider : TimeProviderBase
{
    ...
}

  2. Override the GetTime method. It should return the time value that will be displayed by the AnalogClock. For example, the LocalTimeProvider has the following GetTime method:

    public override TimeSpan GetTime()
{
    return DateTime.Now.TimeOfDay;
}
