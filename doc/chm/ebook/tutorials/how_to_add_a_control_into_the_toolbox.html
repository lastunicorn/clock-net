How to add a control into the toolbox

  1. Right click inside the Toolbox and choose "Add Tab" from the context menu. Name the newly created tab "Clock" (or any other name).
  2. Right click inside the "Clock" tab. Choose "Choose Items..." from the context menu.
  3. In the new dialog, on the ".Net Framework Components" click the "Browse..." button and open the "ClockNet.dll" file that you downloaded.
