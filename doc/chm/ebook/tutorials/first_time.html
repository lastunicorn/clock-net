First Time Using the ClockNet

Overview

To have a fully functional clock on your form you need to do two things: 1) drag the ClockNet control from the Toolbox to your form and 2) attach a Timer to it. But if it is the first time to use the ClockNet, you need to add it to the Toolbox first. So there are three steps:

  1. Add the "ClockNet" control to the Toolbox.
  2. Drag a new "ClockNet" control on your form.
  3. Attach a timer to the clock.

(1) Add the "ClockNet" control to the Toolbox.

  1. Right click inside the Toolbox and choose "Add Tab" from the context menu. Name the newly created tab "Clock" (or any other name).
  2. Right click inside the "Clock" tab. Choose "Choose Items..." from the context menu.
  3. In the new dialog, on the ".Net Framework Components" click the "Browse..." button and open the "ClockNet.dll" file that you downloaded.

Now you should heve in the Toolbox's tab, the "ClockNet" control.

(2) Drag a new "ClockNet" control on your form.

This is simple: Drag from the Toolbox a new "ClockNet" control and drop it on your form.

(3) Attach a timer to the clock.

  1. Drag from the ToolBox on your form a new "Timer". It can be found in the "Components" tab. Set the Interval value and enable it from the "Properties" tool window.
  2. Select the "ClockNet" added on your form. In the "Properties" tool window find the "Timer" property and set it to use the timer created in step 3.

Note! Additionally you may change the properties of the default shapes of the "ClockNet" in "Properties" tool window in the "Shapes" category to make the clock look prettier.

Congratulations! You now have a working clock that displays the current system time.
