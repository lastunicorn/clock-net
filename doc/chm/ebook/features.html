Features

This is the section where I should prise my control, isn't it?

  • Decoupled design. The update is performed by the timer, the displayed value is provided by the Time Provider and the way the value is displayed is controlled by Shapes.
  • Customizable appearance. Using custom implementations of the IShape interface ClockNet can look as you like.
  • The ability to display a custom time (not only the system's time) using a custom implementation of the ITimeProvider.
  • Scalable GUI. You can resize the clock from very small to very large. It will look the same.
  • Easy to use. Add a ClockNet control to your form and attach a timer. That's it. ;)
